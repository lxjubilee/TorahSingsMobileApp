Mobile App Category Order
The mobile app should display the main categories in the following order:
1.	HOME
2.	INSPIRE FAMILY
3.	FAMILY FRIENDLY
4.	CHILDREN MUSIC
5.	MUSIC TYPE
These categories should be manageable through the admin section of the JubiLujah.com website.
The admin section should allow us to control:
•	Whether each category appears in the mobile app
•	The display order of the categories
•	Whether a category is active or inactive
•	Which albums, artists, or collections appear inside each category
•	The display order of albums inside each category
Home Category
The HOME category should be the main landing section of the mobile app.
Home should not automatically display Inspire personas. Instead, Home should display the primary app content that we intentionally select or activate through the admin panel.
The Home section should be controlled from the admin area so we can decide what appears there.
Inspire Family Category
The INSPIRE FAMILY category should display the approved Inspire Family artists/personas only.
Not everything with the Inspire name should automatically appear in this category. For example, Gabriel Inspire should not appear unless we specifically activate it.
We need an admin checkbox such as:
Show in Inspire Family Mobile Section
Only approved Inspire Family artists/personas should appear here.
Family Friendly Category
The FAMILY FRIENDLY category should display approved family-friendly music and albums.
Melody Inspire should probably appear near the top of this section, but the exact content and ordering should be controlled through the admin panel.
We need to be able to activate or deactivate what appears in Family Friendly for the mobile app.
Children Music Category
The CHILDREN MUSIC category should display approved children’s music collections, albums, and artists.
This category should also be controlled through the admin panel.
The admin should allow us to decide:
•	Which children’s albums appear
•	Which artists or collections appear
•	The display order
•	Whether the category is active or inactive in the mobile app
Music Type Category
The MUSIC TYPE category should allow users to browse music by music style or genre.
The Music Type section should list the following music types first, in this order:
1.	Contemporary
2.	Praise & Worship
3.	Country
4.	Pentecostal Shout
5.	Gospel
After those first five music types, the app should display any additional music type that has at least 12 albums or more associated with that music type.
The Music Type section should also be manageable through the admin panel, including:
•	Whether a music type appears in the mobile app
•	The display order of music types
•	The minimum album count requirement
•	Whether a music type is active or inactive
•	Which albums are associated with each music type
Overall Admin Requirement
The JubiLujah.com website admin section needs a dedicated Mobile App Settings area.
This section should allow us to manage the mobile app separately from the website, because some content may be visible on the website but not ready to appear in the mobile app.
The mobile app settings should control categories, artists, albums, music types, display order, visibility, and active/inactive status.
